# Rutherford.fr clone React (modifiable)

## Lancer

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Où modifier le contenu

- Texte et structure: `src/data/siteContent.js`
- Sections UI: `src/components/*`
- Style: `src/styles.css`

## Notes

- Cette version est une base React editable inspirée de la structure observée sur `rutherford.fr`.
- Tu peux remplacer les contenus, couleurs, liens et blocs sans toucher au routing.
